# Changelog - Корпоративный чат

## Версия 1.0.0 (2024-01-XX)

### Добавлено
- Базовая функциональность чата
- Поддержка различных типов медиа (текст, изображения, видео, аудио, документы, стикеры, локация)
- Real-time обновления через Server-Sent Events (SSE)
- Система ролей и доступов (admin, manager)
- Назначение чатов по отделам
- История чатов
- Шаблоны ответов
- Загрузка медиа файлов
- Поиск по чатам
- Модальные окна для создания чатов

### Исправлено
- Улучшена производительность загрузки сообщений
- Исправлены ошибки в отображении медиа файлов
- Оптимизирована работа с базой данных

### Изменено
- Рефакторинг кода на компоненты для лучшей организации
- Улучшен пользовательский интерфейс
- Добавлена поддержка темной темы
